# Meu Currículo em React

Projeto individual desenvolvido como atividade do curso, com o objetivo de criar uma página web de currículo/portfolio utilizando os conceitos fundamentais de React.

## Objetivo do Projeto

Desenvolver uma página de currículo profissional em React, apresentando informações como perfil, competências, formação, projetos, experiências profissionais e idiomas.

O projeto também inclui uma interação entre duas versões do currículo:

- **Versão para Humanos**: apresentação visual, organizada e profissional.
- **Versão para IA**: versão codificada e estruturada do currículo, criada como recurso interativo para demonstrar o uso de `useState`.

> A versão para IA não possui comandos invisíveis, instruções ocultas ou tentativa de manipular sistemas. É apenas uma funcionalidade criativa e interativa do projeto.

## Tecnologias Utilizadas

- React
- Vite
- JavaScript
- HTML
- CSS
- React Icons
- Git e GitHub

## Funcionalidades

- Página de currículo em formato de portfolio.
- Componentização em React.
- Uso de props em componentes reutilizáveis.
- Renderização de listas com `.map()`.
- Interação com `useState`.
- Alternância entre versão humana e versão para IA.
- Estilização com CSS puro.
- Layout responsivo.
- Uso de ícones com React Icons.
